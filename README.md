# Mirvan LLC — mirvaninc.com

A static marketing site for Mirvan LLC, built for deployment on Cloudflare Pages.

## Stack

- Static HTML / CSS / vanilla JS — zero build step (Python only used to generate city pages from JSON)
- Google Fonts: Fraunces (display), Inter Tight (sans), JetBrains Mono (mono)
- Black-and-white editorial aesthetic with light/dark mode

## File structure

```
.
├── index.html                              # homepage
├── styles.css                              # all styles (shared by every page)
├── main.js                                 # theme toggle + scroll reveals
├── favicon.svg
├── robots.txt
├── sitemap.xml                             # auto-generated by build-cities.py
├── _headers                                # Cloudflare caching + security headers
├── cities.json                             # source data for city pages — EDIT THIS to add cities
├── build-cities.py                         # regenerates city pages from cities.json
└── property-management-software/
    ├── index.html                          # hub page listing all cities
    ├── san-diego/index.html
    ├── los-angeles/index.html
    ├── phoenix/index.html
    ├── las-vegas/index.html
    ├── austin/index.html
    ├── dallas-fort-worth/index.html
    ├── miami/index.html
    └── tampa/index.html
```

## Adding or editing a city page

1. Edit `cities.json` (add an entry, change copy, etc.)
2. Run `python3 build-cities.py`
3. Commit and push — Cloudflare Pages deploys automatically

The build script regenerates every city page, the hub page, and the sitemap.

## Features

- Mobile-first responsive layout
- Light/dark mode that follows system preference and can be overridden manually
- No flash of incorrect theme (theme set inline before paint)
- Accessible: skip link, focus styles, semantic HTML, reduced-motion support
- SEO-ready: meta tags, Open Graph, Twitter cards, JSON-LD (Organization, ProfessionalService, BreadcrumbList per city), sitemap.xml, robots.txt
- City pages with `LocalBusiness` schema, `areaServed`, and breadcrumb structured data
- Performant: no JS frameworks, fonts preconnected, immutable cache headers via `_headers`

## Deploying to Cloudflare Pages

1. Push this directory to a new GitHub repo.
2. In Cloudflare → Workers & Pages → Create → Pages → Connect to Git.
3. Select the repo. Build settings:
   - **Framework preset:** None
   - **Build command:** `python3 build-cities.py` (or leave blank if you commit the generated files)
   - **Build output directory:** `/`
   - **Root directory:** `/`
4. Save and Deploy.
5. In the Pages project → Custom domains → add `mirvaninc.com` and `www.mirvaninc.com`.
